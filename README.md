## Description

> This project lets users move within a limited canvas, jump between platforms, and steal items using keyboard inputs.  
> It explores basic object animation, movement, and object interaction like if the player touches a platform or an item.

---

## Interaction Instructions

> Press A/D or left/righ arrow keys to move left or right.
> Press W, space bar,or up arrow key to jump on to platforms and steal items by touching them.
